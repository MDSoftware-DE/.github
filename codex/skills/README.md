# Codex Skills (MDSoftware-DE)

This folder contains reusable skill definitions for Codex instances.

## Available Skill
- `org-agents-governance`

## Install on another Codex instance
1. Copy `codex/skills/org-agents-governance` into that instance under `$CODEX_HOME/skills/`.
2. Keep `SKILL.md` and `references/checklist.md` together.
3. Use the skill when aligning repositories to org AGENTS standards.
